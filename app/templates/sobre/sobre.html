Sobre

Tema Escolhido

O tema escolhido para a aplicação desenvolvida para o trabalho de tópicos especiais, foi a questão de gestão de tarefas, sendo desenvolvido um site onde é possivel criar lembretes, separando estes em murais onde o usuário pode colocar o nome que desejar

Objetivo do Projeto

O objetivo é realizar o desenvolvimento de um site de criação de lembretes, separados por categorias personalizadas, com a linguagem python com a lib flask atuando no back-end e html e css para o front-end, salvando em um banco de dados criado com MySql
